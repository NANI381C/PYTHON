# PYTHON 🐍

A comprehensive collection of Python scripts demonstrating foundational to advanced programming concepts. 

## 📚 About This Repository
This repository serves as a practical reference guide and code vault for Python programming. It contains various scripts categorized by fundamental computer science topics, specifically focusing on Object-Oriented Programming (OOP), Functions, Data Structures, and Error Handling.

## 🗂️ Concepts Covered

### Object-Oriented Programming (OOP)
* **Classes and Objects:** Real-world modeling examples (`class fan.py`, `class student.py`, `class car.py`, `class Bike.py`, etc.)
* **Inheritance:** Base and derived class implementations (`inheritence.py`, `inheritence1.py`)
* **Polymorphism:** Method overloading and overriding (`overloading.py`, `overriding.py`, `plane-polymorphism.py`)
* **Encapsulation:** Standard, property-based, and access modifiers (`encapsulation.py`, `normal encapsulation.py`, `property encapsulation.py`)
* **Relationships:** Composition and aggregation (`compotiton of both Composition and aggrigation.py`)

### Functions & Methods
* **Function Basics:** Various parameter and return type combinations (`8.1` to `8.4` files).
* **Advanced Functions:** * Lambda functions (`lambdafunction.py`)
    * Higher-order functions (`higher ordered function.py`)
    * Decorators (`decorators.py`)
    * Generators (`generator.py`)
* **Arguments:** Default and Keyword arguments (`10.DefaultAndKeyWordArguments.py`), Packing and Unpacking (`packing and unpacking.py`).

### Control Flow
* **Loops & Conditionals:** Core logic structures (`7.1.for loop.py`, `7.2.Conditions.py`).

### Data Structures & Strings
* **String Manipulation:** Slicing (`12.Slicing.py`), interning (`11.stringinterning.py`), formatting, reversing, and case swapping.
* **Lists:** Built-in list functions and list arithmetic.
* **Data Validation:** Using string methods like `isalpha`, `isdigit`, and `isalnum`.

### Exception Handling
* **Error Management:** Try-except blocks and standard error handling (`exceptions.py`, `exceptions2.py`, `printing error message defaultly.py`).
